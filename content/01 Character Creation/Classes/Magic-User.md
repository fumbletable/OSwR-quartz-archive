# Magic-User

**HD:** 4 + CON then 1d4 (2) + CON  
**Saves:** INT, WIS  
**Ability Boost:** +1 INT  
**Weapons:** Light  
**Armor:** None  

**Spellcasting (SP).** INT-based. Save DC = 8 + PB + INT; Spell attack = PB + INT.  
Prepare (level + INT mod) spells; spend SP to cast; recover per [[01 Rules/Rests & Recovery]].  

**Cantrips.** *Wizard Bolt*, *Prestidigitation* at level 1.  

**Boost Hook.** After a successful **Breather SP refresh**, you may spend 1 Boost Die to regain +1 SP,  
or extend a spell’s **duration one step** where sensible (Referee adjudicates).  

**Spellbook.** Starts with *Read Magic* + (1 + INT mod) spells.  
Add spells by finding scrolls/books or research.  

**Starting Kit (pick one):**  
- *Scholar:* Dagger, Arcane Staff, Spellbook.  
- *Apprentice-on-the-Road:* Dagger, Spellbook, Arcane Focus, Chalk & charcoal.
